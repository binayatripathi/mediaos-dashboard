# MediaOS Dashboard

A control room for multi-brand video operations.

## Purpose
Manage:
- brand packs
- workflow routing
- video job creation
- render queue
- approvals
- asset-aware production

## V1
- Dashboard landing page
- Brand pack overview
- Workflow registry
- Create-job panel
- Queue/status view

## Local dev
```bash
npm install
npm run dev
```

## Deploy
Import this repo into Replit and deploy from there.
