# MediaOS Dashboard

A React + Vite frontend for a multi-brand media operating system.

## Purpose
Manage:
- brand packs
- workflow routing
- script generation
- asset generation / upload
- render jobs
- approvals
- publishing
- analytics feedback

## Stack
- React
- Vite
- TypeScript

## Local dev
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy
Import into Replit and deploy from there.
